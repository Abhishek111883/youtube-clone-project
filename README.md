# YouTube Clone - Frontend (HTML & CSS)

This repository contains the frontend code for a YouTube clone project, implemented using HTML and CSS only. The project replicates the essential visual elements of the popular video-sharing platform, YouTube, focusing on frontend development without any JavaScript or backend functionalities.

## Features

- **Responsive Design:** The frontend ensures a responsive layout for optimal viewing on various devices.

- **Video Thumbnails:** Displays video thumbnails with relevant information.

- **User Interface:** Provides a user-friendly interface for browsing videos.

- **Static Content:** The project includes static content and does not involve dynamic data fetching.


## Technologies Used

- **HTML:** For structuring the content and layout.

- **CSS:** For styling the HTML elements.
